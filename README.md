# Daniel Lim - Portfolio Website

Personal portfolio website showcasing my experience, projects, and skills.

**Live at:** [daniellim.net](https://daniellim.net) | [daniellim-website.web.app](https://daniellim-website.web.app)

## Tech Stack

- **Framework:** Next.js 14 (App Router) + TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Firebase Hosting (static export)
- **Analytics:** Firebase Analytics

## Sections

- Hero - Introduction and CTA
- About - Bio, education (UC Berkeley CS, 3.8 GPA)
- Experience - Karass, Kitecyber, Superflow roles
- Projects - Productivity Calendar, Image Classification, Secure File Sharing
- Skills - Languages, Frameworks, Tools, Libraries
- Resume - Embedded PDF viewer with download
- Contact - Contact form + social links

## Development

```bash
npm install
npm run dev        # Start dev server at localhost:3000
```

## Deployment

```bash
npm run build      # Static export to out/
firebase deploy    # Deploy to Firebase Hosting
```
